# 👖 Jackie Jeans – Smart Fit Onboarding

A modern, mobile-first onboarding experience built for the Jackie Jeans Internship Hiring Challenge.

This project provides two onboarding flows to help users find their perfect jeans fit:

* 📝 Manual Onboarding Quiz
* 🎙️ AI Voice Onboarding

Both flows collect the user's fit preferences and redirect them to the Jackie Jeans website after completion.

---

## ✨ Features

### 📝 Manual Onboarding

* Multi-step onboarding experience
* Mobile-first responsive UI
* Progress indicator
* Input validation
* Optional weight question
* Multi-select brand selection
* Dynamic brand-size collection
* Smooth navigation between questions

### 🎙️ AI Voice Onboarding

* Speech synthesis (AI asks questions)
* Speech recognition (User answers by voice)
* Natural conversational experience
* Automatic question progression
* Voice confirmation after each response

---

## 🛠️ Tech Stack

* React
* Vite
* Tailwind CSS
* Framer Motion
* Web Speech API (Speech Recognition + Speech Synthesis)

---

## 📋 Quiz Flow

The onboarding collects:

1. Height
2. Weight (Optional)
3. Waist Measurement
4. Hip Measurement
5. Waist Fit Preference
6. Waistband Position
7. Thigh Fit
8. Previously Worn Brands
9. Brand Sizes
10. Biggest Fit Frustration

## 🚀 Getting Started

Clone the repository:

`git clone https://github.com/your-username/jackie-jeans-onboarding.git`

Install dependencies:

Run the development server:

npm run dev

Build for production:

npm run build
